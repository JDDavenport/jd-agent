## Quick Tips

### Most Used Shortcuts

| Shortcut | Action |
|----------|--------|
| `⌘ K` | Open search / command palette |
| `⌘ N` | Create new task |
| `⌘ Enter` | Quick add task |
| `G T` | Go to Today view |
| `G I` | Go to Inbox |

### Quick Add Examples

```
Buy groceries tomorrow @errands
Review report p1 #work
Call doctor Monday 2pm
```
