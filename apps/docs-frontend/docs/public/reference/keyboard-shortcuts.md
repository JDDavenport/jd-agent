# Keyboard Shortcuts

Quick reference for all keyboard shortcuts in JD Agent.

---

## Global Shortcuts

These work across all apps:

| Shortcut | Action |
|----------|--------|
| `Cmd+K` / `Ctrl+K` | Open search |
| `Esc` | Close modal/panel |
| `?` | Show keyboard shortcuts help |

---

## Tasks App

### Navigation

| Shortcut | Action |
|----------|--------|
| `G then I` | Go to Inbox |
| `G then T` | Go to Today |
| `G then U` | Go to Upcoming |
| `G then P` | Go to Projects |

### Task Management

| Shortcut | Action |
|----------|--------|
| `Q` | Quick add task |
| `N` | New task (detailed form) |

| `Cmd+Enter` | Complete selected task |
| `Cmd+Shift+Enter` | Complete and create next |
| `Delete` / `Backspace` | Delete selected task |
| `E` | Edit selected task |

## Tips

### Learning Shortcuts
1. Start with the most common: `Q`, `Cmd+K`, navigation
2. Print this page as a reference
3. Use `?` to see available shortcuts in context

### Efficiency
1. Use `G then X` navigation to avoid mouse
2. Use quick add for fast capture
3. Learn vim-style `J`/`K` navigation
