# Quick Start Guide

Get productive with JD Agent in 5 minutes.

---

## Your First 5 Minutes

### 1. Open the Tasks App

Navigate to http://localhost:5174

You'll see the main Tasks interface with:
- **Sidebar** - Navigation between views
- **Main area** - Your tasks
- **Quick add** - Task input at the top

---

### 2. Create Your First Task

Click the quick add bar at the top (or press `Q`) and type:

```
Review JD Agent documentation
```

Press **Enter** to create the task.

**Congratulations!** You've created your first task. It's now in your Inbox.

---

### 3. Try Natural Language

Create another task with more details:

```
Call mom tomorrow at 2pm @calls
```

This creates a task that:
- Is titled "Call mom"
- Is scheduled for tomorrow at 2:00 PM
- Has the @calls context

### More Examples

Try these:
```
Buy groceries @errands
Finish report Friday p1
Review PR #Work ~30m
Read chapter 5 someday
```

| Syntax | Meaning |
|--------|---------|
| `tomorrow`, `Friday` | Due date |
| `at 2pm` | Scheduled time |
| `@calls`, `@errands` | Context |
| `p1`, `p2`, `p3` | Priority (1=urgent) |
| `#Work` | Project |
| `~30m`, `~2h` | Time estimate |
| `someday` | Someday/maybe status |

---

### 4. Explore the Views

Click through the sidebar to see different views:

| View | Purpose |
|------|---------|
| **Inbox** | New, unprocessed tasks |
| **Today** | Tasks scheduled for today |
| **Upcoming** | Tasks with future dates |
| **Projects** | Tasks organized by project |

---

### 5. Complete a Task

Click the checkbox next to any task to complete it.

Completed tasks are automatically archived to the Vault, creating a permanent record of your accomplishments.

---

## Your First 15 Minutes

Now let's organize a bit more:

### Create a Project

1. Click **Projects** in the sidebar
2. Click **+ New Project**
3. Enter a name like "Personal" or "Work"
4. Click **Create**

### Move Tasks to Projects

1. Click on a task to open it
2. Click the **Project** dropdown
3. Select your project
4. Close the task

Or use quick add:
```
New task idea #Personal
```

### Add Context to Tasks

Contexts describe WHERE or HOW you can do a task:

| Context | When to Use |
|---------|------------|
| `@computer` | Needs a computer |
| `@calls` | Phone calls to make |
| `@errands` | While out of the house |
| `@home` | At home |
| `@waiting` | Waiting for someone |

Add context when creating:
```
Reply to John's email @computer
```

Or edit an existing task and add contexts.

## Try the AI Agent

### Open Chat

Navigate to the Command Center (http://localhost:5173) and click **Chat**.

### Ask the Agent

Try these commands:

**Create a task:**
```
Create a task to review the quarterly report by Friday
```

**Check your schedule:**
```
What's on my calendar today?
```

**Search your system:**
```
What tasks are due this week?
```

**Add to your knowledge base:**
```
Remember this: My car's license plate is ABC-1234
```

The agent has 37 tools and can help with tasks, calendar, vault, and more.

## Key Concepts to Remember

### The Inbox
- Everything starts in the Inbox
- Process it daily (aim for inbox zero)
- Ask: "What's the next action?"

### Today View
- Your focus for the day
- Includes scheduled tasks + overdue items
- Check this first thing in the morning

### Projects
- Outcomes requiring multiple tasks
- Group related work together
- Each task should have one project

### Contexts
- Filter by WHERE you can work
- @computer when at your desk
- @calls when you can make calls
- @errands when you're out

## Tips for Success

1. **Capture everything** - Get it out of your head into the inbox
2. **Process daily** - Review and organize your inbox
3. **Trust the system** - If it's in JD Agent, you won't forget it
4. **Use contexts** - They help you work efficiently
5. **Complete tasks** - The satisfaction is real!
